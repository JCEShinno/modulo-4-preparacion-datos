# Proyecto Módulo 4 - Preparación de Datos con Python

## Descripción
Proyecto integrador enfocado en la preparación de datos para un caso de e-commerce, desarrollado con Python, NumPy y Pandas. El trabajo contempla generación de datos, integración de múltiples fuentes, limpieza, tratamiento de valores nulos y outliers, data wrangling, agrupamiento, pivoteo y exportación final del dataset.

## Objetivo
Desarrollar un flujo de trabajo integral para obtener, limpiar, transformar y estructurar datos provenientes de distintas fuentes, dejándolos listos para análisis y potencial uso en modelos predictivos.

## Tecnologías utilizadas
- Python
- NumPy
- Pandas
- openpyxl
- lxml

## Estructura del proyecto
- `data/raw/`: datos base y archivos fuente
- `data/interim/`: archivos intermedios generados durante el proceso
- `data/processed/`: dataset final exportado
- `notebooks/`: notebook principal del proyecto
- `outputs/`: archivos resumen
- `docs/`: documentación e informe final

## Flujo de trabajo
1. Generación de datos ficticios con NumPy
2. Exploración inicial con Pandas
3. Integración de fuentes CSV, Excel y HTML
4. Tratamiento de valores nulos y outliers
5. Data Wrangling
6. Agrupamiento, pivoteo y exportación final

## Entregables
- Notebook del proyecto
- Dataset final en CSV
- Dataset final en Excel
- Resumen comercial en CSV
- Informe final del proyecto

## Resultados principales
- Dataset final con 1200 registros y 41 columnas
- Integración de múltiples fuentes de datos
- Corrección de inconsistencias, duplicados y valores atípicos
- Generación de variables derivadas y métricas agregadas

## Cómo ejecutar el proyecto

1. Clonar el repositorio.
2. Crear y activar un entorno virtual.
3. Instalar las dependencias requeridas: `numpy`, `pandas`, `openpyxl` y `lxml`.
4. Abrir el notebook `notebooks/proyecto_modulo_4.ipynb`.
5. Ejecutar las celdas en orden para reproducir el flujo completo de preparación de datos.
